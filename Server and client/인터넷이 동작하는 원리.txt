인터넷이 동작하는 원리

인터넷이 작동하기 위해서는 최소 2대의 컴퓨터가 필요하다.

1개는 웹 브라우저
다른 1개는 웹 서버

이 두대는 인터넷으로 연결되어 있다.

웹서버가 깔려 있는 컴퓨터는 info.cern.ch 라는 주소를 갖고 있으며, 하드 디스크가 있고, index.html이라는 파일이 저장되어 있음
웹브라우저에서 주소창에다가 info.cern.ch/index.html을 쳤을 때 어떤 일이 일어날까

웹 브라우저는 웹 서버에 신호를 보낸다.(client) 이때 이 신호 안에는 index.html을 원합니다. 라는 정보가 담겨 있다.
그러면 웹 서버라는 프로그램은 하드 디스크에서 index.html이라는 파일을 찾아서 웹 브라우저가 있는 컴퓨터에게 쏴준다.(server)


이렇게 둘은 서로 정보를 주고 받는다. 우리가 자주 쓰는 것 -> 웹브라우저

지금부터 할 작업들은 웹 서버를 사용하여 내 컴에 있는 문서를 인터넷이 연결되어 있는 다른 컴퓨터에 보내는 일을 하는 것이다.

웹 서버는

1. web hosting -> 회사에 맡기는 것, 쉬워, 그래서 어떤 동작원리인지 우리가 알 수 없어
2. web server -> 웹 서버를 자신의 컴퓨터에 직접 설치하여 사용하는 방식, 어려워, 근데 그 과정에서 인터넷에 대해 공부를 많이 할 수 있어
