웹 호스팅

인터넷에 연결된 컴퓨터 하나 하나 : host, cloud
이를 빌려주는 사업을 hosting 사업

web hosting 업체: 웹 서버를 전문적으로 빌려주는 비즈니스

수 많은 웹 호스팅 업체가 있음
그 중에서 무료이고, 유명한 서비스인 깃허브의 페이지 기능을 사용할 것임

이 서비스는 프로그래머들에게는 성지와 같은 중요한 서비스임
전 세계의 수 많은 오픈 소스들이 이곳에서 만들어지고 있음

우리는 github가 제공하는 여러 기능 중 웹 호스팅 기능을 이용해서 우리의 홈페이지를 운영할 것임
