웹페이지를 만드는 방법과 만든 html 파일을 여는 방법에 대해 기술되어 있었다.  

웹페이지를 만드는 방법:  
1. visual studio를 킨다.  
2. 폴더 안에 파일을 만들고 확장자 명을 "html"로 한다.  


만든 html 파일을 여는 방법:  
1. ctrl + O 를 눌러 아까 만든 폴더에 들어가 파일을 클릭한다.  
2. 그냥 크롬에 드래그하고 엔터친다.  

끝!
